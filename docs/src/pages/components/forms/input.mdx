# Input Component

The Input Component is a highly versatile and customizable component. It supports various input types, including `text`, `email`, `date`, `file`, `password`, `search`, and `number`. The component is pre-styled with a variety of classes for different sizes, labels, help texts, states, icons, nested select elements, and color themes.

## Features

- Pre-styled using `.input` class
- Larger size using `.input-large`
- Supports labels using the `label` element and `.label` class
- Supports help text using the `label` element and `.helper` class
- Supports states such as success or error using the `.success` or `.error` class on the `span` element
- Supports disabled state using the `disabled` property on the input
- Supports icons either before or after the input component using `svg` or `img`
- Supports a nested select element either before or after the input using `.input-select` class
- Supports prefix and suffix using a span with either `.input-prefix` or `.input-suffix` class either before or after the input element to create a grouped input
- Can use `.input-[any tailwind color name here]` to change the accent color theme
- Can customise the default color, fontweight, text transform and radius in the tailwind.config.js file

## Class Names, Description, and Type

| Class Name                              | Description                                                       | Type       |
| --------------------------------------- | ----------------------------------------------------------------- | ---------- |
| `.input`                                | Provides basic styles for the input component                     | Component  |
| `.input-large`                          | Applies larger size to the input component                        | Modifier   |
| `.label`                                | Styles the label for the input                                    | Component  |
| `.helper`                               | Styles the help text for the input                                | Component  |
| `.success` or `.error`                  | Styles the success and error states for the input                 | State      |
| `disabled`                              | Applies the disabled state to the input                           | State      |
| `.input-select`                         | Styles a nested select element either before or after the input   | Component  |
| `.input-prefix` or `.input-suffix`      | Styles a prefix or suffix for the input to create a grouped input | Modifier   |
| `.input-[any tailwind color name here]` | Changes the accent color theme of the input                       | Responsive |

Please refer to the class names and their types listed above to apply the desired styles to your accordion component.
