# Textarea Component

The Textarea Component is a flexible and customizable component pre-styled for your convenience. It supports labels, help text, different states, and a disabled state. The accent color theme can be easily changed, and you can customize the default color, font weight, text transform, and radius in the tailwind.config.js file.

## Features

- Pre-styled using `.textarea` class
- Larger size using `.textarea-large`
- Supports labels using the `label` element and `.label` class
- Supports help text using the `label` element and `.helper` class
- Supports states such as success or error using the `.success` or `.error` class on the `textarea` element
- Supports disabled state using the `disabled` property on the `textarea`
- Can use `.textarea-[any tailwind color name here]` to change the accent color theme
- Can customize the default color, font weight, text transform, and radius in the tailwind.config.js file

## Class Names, Description, and Type

| Class Name                                 | Description                                          | Type       |
| ------------------------------------------ | ---------------------------------------------------- | ---------- |
| `.textarea`                                | Provides basic styles for the textarea component     | Component  |
| `.textarea-large`                          | Applies larger size to the textarea component        | Modifier   |
| `.label`                                   | Styles the label for the textarea                    | Component  |
| `.helper`                                  | Styles the help text for the textarea                | Component  |
| `.success` or `.error`                     | Styles the success and error states for the textarea | State      |
| `disabled`                                 | Applies the disabled state to the textarea           | State      |
| `.textarea-[any tailwind color name here]` | Changes the accent color theme of the textarea       | Responsive |
