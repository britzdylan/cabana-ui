# Checkbox Component

The Checkbox Component is a flexible, customizable component that has been pre-styled for your convenience. It supports custom icons, a disabled state, and changing the accent color theme. Additionally, you can customize the default color, font weight, text transform, and radius in the tailwind.config.js file.

## Features

- Pre-styled using `.checkbox` class
- Larger size using `.checkbox-large`
- Supports custom icons
- Supports disabled state using the `disabled` property on the checkbox
- Can use `.checkbox-[any tailwind color name here]` to change the accent color theme
- Can customize the default color, font weight, text transform, and radius in the tailwind.config.js file

## Class Names, Description, and Type

| Class Name                                 | Description                                      | Type       |
| ------------------------------------------ | ------------------------------------------------ | ---------- |
| `.checkbox`                                | Provides basic styles for the checkbox component | Component  |
| `.checkbox-large`                          | Applies larger size to the checkbox component    | Modifier   |
| `disabled`                                 | Applies the disabled state to the checkbox       | State      |
| `.checkbox-[any tailwind color name here]` | Changes the accent color theme of the checkbox   | Responsive |
