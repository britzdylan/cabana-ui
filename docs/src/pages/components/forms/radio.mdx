# Radio Component

The Radio Component is a flexible, customizable component that has been pre-styled for your convenience. It supports disabled state, and changing the accent color theme. Additionally, you can customize the default color, font weight, text transform, and radius in the tailwind.config.js file.

## Features

- Pre-styled using `.radio` class
- Larger size using `.radio-large`
- Supports disabled state using the `disabled` property on the radio
- Can use `.radio-[any tailwind color name here]` to change the accent color theme
- Can customize the default color, font weight, text transform, and radius in the tailwind.config.js file

## Class Names, Description, and Type

| Class Name                              | Description                                   | Type       |
| --------------------------------------- | --------------------------------------------- | ---------- |
| `.radio`                                | Provides basic styles for the radio component | Component  |
| `.radio-large`                          | Applies larger size to the radio component    | Modifier   |
| `.disabled`                             | Applies the disabled state to the radio       | State      |
| `.radio-[any tailwind color name here]` | Changes the accent color theme of the radio   | Responsive |
