# Toggle Component

The Toggle Component is a flexible, customizable component that has been pre-styled for your convenience. It supports disabled state, and changing the accent color theme. Additionally, you can customize the default color, font weight, text transform, and radius in the tailwind.config.js file.

## Features

- Pre-styled using `.toggle` class
- Larger size using `.toggle-large`
- Supports disabled state using the `disabled` property on the toggle
- Can use `.toggle-[any tailwind color name here]` to change the accent color theme
- Can customize the default color, font weight, text transform, and radius in the tailwind.config.js file

## Class Names, Description, and Type

| Class Name                               | Description                                    | Type       |
| ---------------------------------------- | ---------------------------------------------- | ---------- |
| `.toggle`                                | Provides basic styles for the toggle component | Component  |
| `.toggle-large`                          | Applies larger size to the toggle component    | Modifier   |
| `.disabled`                              | Applies the disabled state to the toggle       | State      |
| `.toggle-[any tailwind color name here]` | Changes the accent color theme of the toggle   | Responsive |
