# Range Component

The Range Component is a flexible, customizable component that has been pre-styled for your convenience. It supports disabled state, and changing the accent color theme. Additionally, you can customize the default color, font weight, text transform, and radius in the tailwind.config.js file.

## Features

- Pre-styled using `.range` class
- Supports disabled state using the `disabled` property on the range
- Can use `.range-[any tailwind color name here]` to change the accent color theme
- Can customize the default color, font weight, text transform, and radius in the tailwind.config.js file

## Class Names, Description, and Type

| Class Name                              | Description                                   | Type       |
| --------------------------------------- | --------------------------------------------- | ---------- |
| `.range`                                | Provides basic styles for the range component | Component  |
| `.range-large`                          | Applies larger size to the range component    | Modifier   |
| `.disabled`                             | Applies the disabled state to the range       | State      |
| `.range-[any tailwind color name here]` | Changes the accent color theme of the range   | Responsive |
