# Rating Component

The Rating Component is a flexible and customizable component pre-styled for your convenience.

## Features

- Pre-styled using `.rating` class
- Can use `.rating-[any tailwind color name here]` to change the accent color theme
- Used with radio buttons
- Add `checked` attribute to radio button to set the default value
- Defaults to 5 stars

## Class Names, Description, and Type

| Class Name                               | Description                                    | Type      |
| ---------------------------------------- | ---------------------------------------------- | --------- |
| `.rating`                                | Provides basic styles for the rating component | Component |
| `.rating-[any tailwind color name here]` | Changes the accent color theme of the rating   | Modifier  |
