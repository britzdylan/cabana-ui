# Tooltip Component

The Tooltip Component is a flexible and customizable component pre-styled for your convenience. It can be used to display a tooltip on hover.

## Features

- Pre-styled using `.tooltip` class
- Can be added to any element
- The display of the tooltip can be controlled using `data-open` attribute
- The position of the tooltip can be controlled using `data-position` attribute
- The content of the tooltip can be controlled using `data-tooltip` attribute
- Can use `.tooltip-[any tailwind color name here]` to change the accent color theme

## Class Names, Description, and Type

| Class Name                                | Description                                 | Type      |
| ----------------------------------------- | ------------------------------------------- | --------- |
| `.tooltip`                                | Provides basic styles for the tag component | Component |
| `.tooltip-[any tailwind color name here]` | Changes the accent color theme of the tag   | Modifier  |
