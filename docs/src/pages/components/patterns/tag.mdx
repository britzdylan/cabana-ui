# Tag Component

The Tag Component is a flexible and customizable component pre-styled for your convenience. It supports rounded chips, bordered chips, and icon integration. You can also change the accent color theme.

## Features

- Pre-styled using `.tag` class
- Rounded using the `.tag-rounded` class
- Supports bordered version using the `.tag-bordered`
- Supports large version using the `.tag-large` class
- Supports icons
- Can use `.tag-[any tailwind color name here]` to change the accent color theme

## Class Names, Description, and Type

| Class Name                            | Description                                  | Type      |
| ------------------------------------- | -------------------------------------------- | --------- |
| `.tag`                                | Provides basic styles for the tag component  | Component |
| `.tag-rounded`                        | Applies rounded corners to the tag component | Modifier  |
| `.tag-bordered`                       | Applies border to the tag component          | Modifier  |
| `.tag-large`                          | Applies large size to the tag component      | Modifier  |
| `.tag-[any tailwind color name here]` | Changes the accent color theme of the tag    | Modifier  |
