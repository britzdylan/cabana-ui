# Dot Component

The Dot Component is a flexible and customizable component pre-styled for your convenience.

## Features

- Pre-styled using `.dot` class
- Can use `.dot-[any tailwind color name here]` to change the accent color theme

## Class Names, Description, and Type

| Class Name                            | Description                                 | Type      |
| ------------------------------------- | ------------------------------------------- | --------- |
| `.dot`                                | Provides basic styles for the dot component | Component |
| `.dot-[any tailwind color name here]` | Changes the accent color theme of the dot   | Modifier  |
