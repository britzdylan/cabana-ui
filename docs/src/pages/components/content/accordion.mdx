# Accordion

The Accordion component is a pre-styled and highly customizable UI element for your web projects. It's built using Tailwind CSS and offers a set of modifier classes for different styles and behaviors.

## Features

The Pill component includes the following features:

Pre-styled : The Accordion component comes pre-styled with a `.accordion` class.
Open State : Use the `.accordion-open` class to control the open state of the accordion.
Nested Content : Use the `.accordion-body` class for any nested content within the accordion.
Color Customization : The color of the accordion can be modified using `.accordion-[color]` where `[color]` can be any Tailwind CSS color name.
Additional Customizations : The default color, font weight, text transform, and radius can be customized in the tailwind.config.js file.

## Class Reference

Below is a table of the available classes, their purposes, and their types:

| Class Name         | Description                                                                               | Type      |
| ------------------ | ----------------------------------------------------------------------------------------- | --------- |
| .accordion         | The base class for the accordion component                                                | Component |
| .accordion-open    | Controls the open state of the accordion                                                  | Modifier  |
| .accordion-body    | Used for nested content within the accordion                                              | Component |
| .accordion-[color] | Changes the color of the accordion. Replace [color] with any Tailwind CSS color name      | Modifier  |
| tailwind.config.js | Configuration file for customizing default color, font weight, text transform, and radius | Config    |

Please refer to the class names and their types listed above to apply the desired styles to your accordion component.
