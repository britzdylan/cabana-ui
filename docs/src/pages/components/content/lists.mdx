# Lists

The List Component is a versatile and customizable list UI element designed using Tailwind CSS. This component allows you to create and style list items with ease, offering a set of modifier classes for additional styles and behaviors.

## Features

- Pre-styled: The List Component comes pre-styled with a `.list` class.
- Featured Item: Use the `.featured` class to highlight a particular list item.
- Color Customization: The color of the list can be modified using `.list-[color]` where `[color]` can be any Tailwind CSS color name.
- Additional Customizations: The default color, font weight, text transform, and radius can be customized in the tailwind.config.js file..

## Class Reference

Below is a table of the available classes, their purposes, and their types:

| Class Name         | Description                                                                               | Type      |
| ------------------ | ----------------------------------------------------------------------------------------- | --------- |
| .list              | The base class for the list component                                                     | Component |
| .featured          | Marks a list item as featured                                                             | Modifier  |
| .list-[color]      | Changes the color of the list. Replace [color] with any Tailwind CSS color name           | Modifier  |
| tailwind.config.js | Configuration file for customizing default color, font weight, text transform, and radius | Config    |

Please refer to the class names and their types listed above to apply the desired styles to your list component.
