# Toast Component

The Toast Component is a flexible and customizable component pre-styled for your convenience. It can be used to display notifications, alerts, and other messages to the user.

## Features

- Pre-styled using `.toast` class
- Supports position and visibility toggle
- Supports icons
- Can use `.toast-[any tailwind color name here]` to change the accent color theme

## Class Names, Description, and Type

| Class Name                              | Description                                   | Type      |
| --------------------------------------- | --------------------------------------------- | --------- |
| `.toast`                                | Provides basic styles for the toast component | Component |
| `.toast-tl`                             | Moves the toast into the top left corner.     | Modifier  |
| `.toast-tr`                             | Moves the toast into the top right corner.    | Modifier  |
| `.toast-bl`                             | Moves the toast into the bottom left corner.  | Modifier  |
| `.toast-br`                             | Moves the toast into the bottom right corner. | Modifier  |
| `.toast-[any tailwind color name here]` | Changes the accent color theme of the toast   | Modifier  |
| data-show                               | Toggles the visibility of the toast component | Attribute |
