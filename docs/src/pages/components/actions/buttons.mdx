# Button Component

The Button component provides pre-styled buttons with various customization options, including sizing, variants, icon support, color customization, and default style configuration. The Button component is a versatile and customizable button element that can be easily integrated into your web projects. It offers a range of features to suit different design requirements, allowing you to create buttons with different styles, sizes, and behaviors.

## Features

The Button component includes the following:

- Pre-styled using the `.btn` class for quick and consistent styling.
- Sizable variations with classes `.btn-sm`, `.btn-md`, and `.btn-lg` to adjust the button size.
- Variants include Contained, Outline, Minimal, and Text styles, achieved with classes `.btn`, `.btn-outline`, `.btn-min`, and `.btn-text`, respectively.
- Compatible with icons. Add any SVG or any other element using the `.icon` class, positioning them either as the first or last element in the button.
- Ability to change the default color of the button by using `.btn-[color]` with any valid Tailwind CSS color name.
- Customizable default styles such as color, font weight, text transform, and radius in the tailwind.config.js file.(coming soon)
- Disabled state with the `disabled` attribute.
- Icon button styles with the `.fab` class.
- Icon button sizes with the `.fab-sm`, `.fab-md`, `.fab-lg`, and `.fab-xl` classes.
- Icon button rounded styles with the `.fab-rounded` class.
- Icon button outline styles with the `.fab-outline` class.
- Ability to change the default color of the icon button by using `.fab-[color]` with any valid Tailwind CSS color name.
- Button group styles with the `.btn-group` class.
- Button group sizes with the `.btn-group-small` and `.btn-group-large` classes.
- Button group outline styles with the `.btn-group-outline` class.
- Ability to change the default color of the button group by using `.btn-group-[color]` with any valid Tailwind CSS color name.

## Class Reference

Below is a table of the available classes, their purposes, and their types:

| Class                      | Type      | Description                                                                                           |
| -------------------------- | --------- | ----------------------------------------------------------------------------------------------------- |
| .btn                       | Component | Applies pre-styled button styles.                                                                     |
| .btn-sm                    | Modifier  | Makes the button small-sized.                                                                         |
| .btn-lg                    | Modifier  | Makes the button large-sized.                                                                         |
| .btn-outline               | Modifier  | Applies an outline style to the button.                                                               |
| .btn-min                   | Modifier  | Applies a minimal style to the button.                                                                |
| .btn-text                  | Modifier  | Applies a text-only style to the button.                                                              |
| .icon                      | Component | Allows you to include any element within the button to act as an icon.                                |
| .btn-[color]               | Modifier  | Changes the default color of the button. Replace [color] with any valid Tailwind CSS color name.      |
| .btn-outline-[color]       | Modifier  | Changes the default color of the outline button.                                                      |
| .btn-min-[color]           | Modifier  | Changes the default color of the minimal button.                                                      |
| .btn-text-[color]          | Modifier  | Changes the default color of the text button.                                                         |
| .fab                       | Component | Applies pre-styled button styles for icon buttons. Requires additional classes.                       |
| .fab-sm                    | Modifier  | Makes the icon button small-sized.                                                                    |
| .fab-lg                    | Modifier  | Makes the icon button large-sized.                                                                    |
| .fab-xl                    | Modifier  | Makes the icon button extra-large-sized.                                                              |
| .fab-rounded               | Modifier  | Applies a rounded style to the icon button.                                                           |
| .fab-outline               | Modifier  | Applies an outline style to the icon button.                                                          |
| .fab-[color]               | Modifier  | Changes the default color of the icon button. Replace [color] with any valid Tailwind CSS color name. |
| .fab-outline-[color]       | Modifier  | Changes the default color of the outline icon button.                                                 |
| .btn-group                 | Component | Applies pre-styled button group styles.                                                               |
| .btn-group-small           | Modifier  | Makes the button group small-sized.                                                                   |
| .btn-group-large           | Modifier  | Makes the button group large-sized.                                                                   |
| .btn-group-outline         | Modifier  | Applies an outline style to the button group.                                                         |
| .btn-group-[color]         | Modifier  | Changes the default color of the group.                                                               |
| .btn-group-outline-[color] | Modifier  | Changes the default color of the outline group.                                                       |

Please refer to the class names and their types listed above to apply the desired styles to your buttons.

<Alert type='warning'>
  Please do not nest any svg icons inside of a wrapper element that has a class
  of `.icon`, this will cause spacing issues. Only add the `.icon` class to
  elements other than a svg.
</Alert>
