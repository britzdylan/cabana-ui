# Tab Component

The Tab component provides pre-styled tabs with various customization options, including active indicators, icon compatibility, color customization, variants, sizes, and active states.

## Description

The Tab component is used to organize and display content in a tabbed layout, allowing users to switch between different sections of content efficiently. It offers a range of features to style the tabs according to your design requirements and enhance the user experience.

## Features

The Tab component includes the following features:

- Pre-styled using the `.tabs` class for quick and consistent styling.
- Active indicator using the `.active` class, highlighting the currently active tab button.
- Compatible with icons. Add any SVG or other elements as the first or last element within each tab button.
- Ability to change the default color of the tabs by using `.tabs-[color]` with any valid Tailwind CSS color name.
- Variants available to customize the style and behavior of the tabs:
- Navigation variant: Use the `.tabs-navigation` class for a traditional navigation-style tab layout.
- Mobile variant: Use the `.tabs-mobile` class to display the tabs in a mobile-friendly stacked layout.
- Sizes available to adjust the size of the tabs:
- Default size: No additional class required.
- Large size: Use the `.tabs-large` class for larger tabs.
- Active states to further customize the appearance of the active tab:
- Bordered state: Use the `.tabs-bordered` class to add a border to the active tab button.
- Background state: Use the `.tabs-background` class to highlight the background of the active tab button.
- Customizable default styles such as color, font weight, text transform, and radius in the tailwind.config.js file.

## Class Reference

Below is a table of the available classes, their purposes, and their types:

| Class Name       | Type      | Description                                                                                    |
| ---------------- | --------- | ---------------------------------------------------------------------------------------------- |
| .tabs            | Component | Applies pre-styled tab styles.                                                                 |
| .active          | Modifier  | Applies an active style to the tab button, indicating the currently active tab.                |
| .icon            | Component | Allows you to include an icon element within each tab button.                                  |
| .tabs-[color]    | Modifier  | Changes the default color of the tabs. Replace [color] with any valid Tailwind CSS color name. |
| .tabs-navigation | Modifier  | Applies a traditional navigation-style layout to the tabs.                                     |
| .tabs-mobile     | Modifier  | Displays the tabs in a stacked layout for mobile devices.                                      |
| .tabs-large      | Modifier  | Applies a larger size to the tabs.                                                             |
| .tabs-bordered   | Modifier  | Adds a border to the active tab button.                                                        |
| .tabs-background | Modifier  | Highlights the background of the active tab button.                                            |

Please refer to the class names and their types listed above to apply the desired styles to your tab component.
