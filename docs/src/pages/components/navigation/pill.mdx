# Pill Component

The Pill component provides pre-styled pill-shaped elements with various customization options, including variants, icon compatibility, color customization, and default style configuration.

## Description

The Pill component is a versatile and customizable element that can be used to create pills, tags, badges, or any other rounded shape in your web projects. It offers a range of features to style the pills according to your design requirements and enhance the visual appeal of your website or application.

## Features

The Pill component includes the following features:

- Pre-styled using the .pill class for quick and consistent styling.
- Variants include Simple and Active styles, achieved with the .pill and .pill-active classes, respectively.
- Compatible with icons. Add any SVG or other elements using the .icon class, positioning them either as the first or last element within the pill.
- Ability to change the default color of the pill by using .pill-[color] with any valid Tailwind CSS color name.
- Customizable default styles such as color, font weight, text transform, and radius in the tailwind.config.js file.

## Class Reference

Below is a table of the available classes, their purposes, and their types:

| Class Name    | Type      | Description                                                                                    |
| ------------- | --------- | ---------------------------------------------------------------------------------------------- |
| .pill         | Component | Applies pre-styled pill styles.                                                                |
| .pill-active  | Modifier  | Applies an active style to the pill.                                                           |
| .icon         | Component | Allows you to include an icon element within the pill.                                         |
| .pill-[color] | Modifier  | Changes the default color of the pill. Replace [color] with any valid Tailwind CSS color name. |

Please refer to the class names and their types listed above to apply the desired styles to your pill component.
