## Install the package

<Code id='install' client:load>
```bash
npm i cabana-ui

# or

yarn add cabana-ui
```
</Code>

## Add the plugin to your Tailwind config

<Code id='config' client:load>
```js
// tailwind.config.js
module.exports = {
  plugins: [require('cabana-ui')],
};
```
</Code>
